This is my first time in Web design and my 2nd year in Graphic

The home page of my website is built using proper HTML structure with a header, navigation bar, main content area, and footer. At the top, the header contains the site title, and the navigation provides links to other pages such as “About,” “Favorite Things,” and “Contact.” The main section includes a hero area built with "div" elements, featuring a large title, an introductory paragraph, and an image with descriptive alt text. The page also demonstrates a simple two-column layout, with the hero text floated next to the image. This design helps organize the content in a clean and professional way while practicing the use of floats, margins, and padding for spacing. To style the page, embedded CSS is written inside the "style" tags in the "head" section. The CSS sets consistent body fonts and background colors, applies margins and padding to separate content, and adds hover effects for the navigation links. Below the hero section, the layout includes three distinct columns, each with a title, short description, image, and a link to its respective page. This section uses CSS to space the columns evenly and highlight them when hovered over. Overall, the page demonstrates the basics of structure, layout, and styling in Dreamweaver using HTML and CSS.
